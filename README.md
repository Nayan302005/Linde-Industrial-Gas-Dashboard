# 📌 Project Overview

This project presents a comprehensive **Power BI Dashboard** developed for analyzing industrial gas business performance at **Linde**.

The dashboard transforms raw operational data into actionable insights through interactive visualizations, helping stakeholders monitor performance, identify trends, and make data-driven decisions.

---

# ✨ Dashboard Highlights

✅ Executive Summary View

✅ Regional Performance Analysis

✅ Industry-wise Consumption Analysis

✅ Gas Type Distribution Analysis

✅ KPI Tracking & Monitoring

✅ Interactive Filtering with Slicers

✅ Trend Analysis & Forecast Insights

✅ Waterfall Analysis

✅ Comparative Business Metrics

---

# 🎯 Business Objectives

The dashboard was designed to answer key business questions:

- Which region generates the highest revenue?
- Which industries consume the most gas?
- What are the top-performing gas categories?
- How do sales trends change over time?
- Which factors contribute to revenue growth and decline?
- How can decision-makers optimize business performance?

---

# 📊 Dashboard Features

## Executive Dashboard

- Revenue KPIs
- Total Orders
- Total Volume
- Business Growth Metrics
- Performance Indicators

---

## Interactive Visualizations

### 📈 Line Chart
Tracks trends and business performance over time.

### 📊 Clustered Bar Chart
Compares performance across multiple dimensions.

### 🌳 Treemap
Visualizes contribution by industry and gas type.

### 🌊 Waterfall Chart
Shows increases, decreases, and final business impact.

### 📋 Data Table
Provides detailed business records for deeper analysis.

# 🎛️ Slicers Used

The dashboard includes dynamic filters:

- Date
- Region
- Industry
- Gas Type

These slicers allow users to perform advanced data exploration in real time.
